# Changelog

Todos los cambios importantes de este proyecto serán documentados en este archivo.

## [1.0.0] - 2024-04-11

### 🎉 Inicial Release

#### ✨ Características
- **Ruta Optimizada de 3 Días** - Recorrido completo desde Calvià
- **Mapa Interactivo** - Visualización dinámica con Leaflet
- **Diseño Responsive** - Totalmente adaptado a móviles, tablets y escritorio
- **Itinerarios Detallados** - Horarios, restaurantes, consejos por cada parada
- **Información Completa** - Descripción de cada lugar de interés
- **Navegación Fluida** - Cambio fácil entre días
- **Service Worker** - Funcionalidad offline
- **PWA Ready** - Instalable como app en móviles

#### 📁 Archivos Incluidos
- `index.html` - Página principal
- `styles.css` - Estilos responsive
- `script.js` - Lógica interactiva y mapas
- `manifest.json` - Configuración PWA
- `sw.js` - Service Worker
- `robots.txt` - SEO
- `sitemap.xml` - Mapa del sitio
- `package.json` - Dependencias Node
- `.env.example` - Variables de entorno
- `.htaccess` - Configuración Apache
- Documentación completa

#### 🗺️ Rutas Incluidas
1. **Día 1: Caló des Moro - Palma**
   - Caló des Moro: 40m cala virgen
   - Castillo de Bellver: vistas panorámicas
   - Centro histórico de Palma

2. **Día 2: Es Trenc - Sierra Tramontana**
   - Es Trenc: 2km arena blanca
   - Valldemossa: pueblo medieval
   - Deià: pueblo de artistas
   - Mirador Sa Foradada: atardecer

3. **Día 3: Tranvía - Tramontana - Cap Formentor**
   - Fornalutx: pueblo encantador
   - Puerto de Sóller: pescadores y barcos
   - Tranvía vintage 1911: campos de naranjas
   - Cap de Formentor: acantilados +300m
